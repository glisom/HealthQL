# Contributing

Thank you for your interest in contributing to HealthQL!

For full contributing guidelines, see [CONTRIBUTING.md](https://github.com/glisom/HealthQL/blob/main/CONTRIBUTING.md) on GitHub.

## Quick Start

1. Fork the repository
2. Clone your fork
3. Run tests: `swift test`
4. Make your changes
5. Submit a pull request

## Development

### Running Tests

```bash
swift test
```

### Code Style

- Follow Swift naming conventions
- Add documentation for public APIs
- Write tests for new features

### Commit Messages

Use conventional commits:

```
feat(parser): add HAVING clause support
fix(executor): handle empty results correctly
docs: update sleep examples
test: add edge case for date parsing
```

## Need Help?

- Open a [GitHub Issue](https://github.com/glisom/HealthQL/issues)
- Check existing issues before creating new ones
